
# NSE AI ML SaaS Scanner

## Features
- Full NSE universe auto-fetch
- AI Score Engine
- Logistic Regression + XGBoost Probability Model
- Multi-user login (basic)
- Render-ready deployment

## Deploy on Render
Build Command:
pip install -r requirements.txt

Start Command:
streamlit run app.py --server.port $PORT --server.address 0.0.0.0
